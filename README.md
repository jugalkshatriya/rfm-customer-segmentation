# Customer Segmentation using RFM Analysis

## Objective
To segment customers based on purchasing behavior and identify high-value, at-risk, and dormant users.

## Dataset
Synthetic ecommerce transaction dataset generated using Python.

## Methodology
1. Calculated Recency (days since last purchase)
2. Calculated Frequency (number of transactions)
3. Calculated Monetary value (total spend)
4. Assigned quartile-based scores (R, F, M)
5. Combined scores to form RFM segments

## Customer Segments
- High Value
- Potential Loyalists
- At Risk
- Lost

## Business Value
- Enables targeted marketing campaigns
- Identifies high-value dormant customers
- Improves retention strategy precision

## Tools Used
Python, Pandas, NumPy
